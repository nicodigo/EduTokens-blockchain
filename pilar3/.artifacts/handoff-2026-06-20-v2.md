## Bugs identificados y fixes aplicados

Todos los fixes están **implementados en el código fuente** pero **no deployados** — la sesión terminó antes del rebuild + push.

| Bug | Causa raíz | Fix | Archivos afectados |
|---|---|---|---|
| **NCT `StreamLostError`** | `publish_mining_task`/`basic_get` sin try/except → thread crashea si RabbitMQ cierra conexión por idle | try/except + `_ensure_rabbitmq_alive` + retry | `nct/nct.py` (+32/−14) |
| **Worker `MinerError: empty stdout` → requeue infinito** | CUDA binary crashea (stderr oculto en DEBUG). `_on_task` requeuea todo | stderr a ERROR, `_parse_miner_stdout` incluye stderr, dead-letter en MinerError | `miner/miner.py` + `worker/worker.py` |
| **Pool `PRECONDITION_FAILED` loop** | Reconnect code setea `auto_ack=True` (debe ser `False`) → doble ack al recibir resultados | `auto_ack=True` → `False` | `pool/pool.py:659` |
| **`send_test_tx.py earn` nonce=0 siempre** | `cmd_earn` no consultaba nonce del account (vs `cmd_spend` sí) | Agregar query `/account/{pubkey}` | `tools/send_test_tx.py` |
| **`GET /account/{pubkey}` 500** | `get_discarded_txns` llama `.decode()` en strings (Redis con `decode_responses=True`) | `list(raw)` en vez de `[tid.decode() ...]` | `storage/chain_store.py:241` |

## Pasos siguientes

### 1. Rebuild + push (prioridad máxima)

```bash
cd pilar2
docker build -f nct/Dockerfile -t us-central1-docker.pkg.dev/edutokens-2026/edutokens-repo/nct:latest . && docker push $_
docker build -f pool/Dockerfile -t us-central1-docker.pkg.dev/edutokens-2026/edutokens-repo/pool:latest . && docker push $_
docker build -f ../pilar3/docker/worker-gpu.Dockerfile -t us-central1-docker.pkg.dev/edutokens-2026/edutokens-repo/worker-gpu:latest .. && docker push $_
```

### 2. Redeploy

```bash
kubectl -n blockchain rollout restart deployment nct
kubectl -n blockchain rollout restart deployment pool-a
kubectl -n g-compumundo rollout restart deployment worker-gpu
```

### 3. Limpiar colas RabbitMQ

```bash
kubectl -n infra exec rabbitmq-0 -- rabbitmqctl purge_queue pool.pool-a.inbox
kubectl -n infra exec rabbitmq-0 -- rabbitmqctl purge_queue pool.pool-a.results
kubectl -n infra exec rabbitmq-0 -- rabbitmqctl purge_queue pool.pool-a.tasks
```

### 4. Verificar

```bash
kubectl -n blockchain port-forward svc/nct 8080:8080 &
curl localhost:8080/health
uv run python ./pilar2/tools/send_test_tx.py earn <authority_privkey> <student_pubkey> 1000 test
```
